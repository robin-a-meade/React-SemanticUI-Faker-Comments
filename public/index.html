React useState useEffect Express Faker Comments Template

This example illustrates the following:

  • A React functional component that fetches data using axios
    • For this it uses the useState and useEffect hooks
  • Rendering an array into React components using map() embedded in JSX
    • https://reactjs.org/docs/lists-and-keys.html#rendering-multiple-components
    • https://www.pluralsight.com/guides/how-to-implement-a-component-%22loop%22-with-react
  • express HTTP server to serve the static files and to handle the route that provides the comment data
  • @faker-js/faker for generating fake data
  • day.js for formatting relative time
  • props.children for handling nested child JSX components
